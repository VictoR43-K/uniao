# Uniao Group Limited - Modern Premium Website

A modern, premium replica of the Uniao Group Limited website featuring cutting-edge design and interactive elements.

## 🌟 Features

### Modern Design
- **Responsive Layout** - Fully responsive design that works on all devices
- **Dark/Light Mode** - Theme toggle with persistent storage
- **Smooth Animations** - Scroll animations, hover effects, and transitions
- **Premium UI/UX** - Clean, professional interface with modern aesthetics

### Interactive Elements
- **Product Filtering** - Filter products by category with smooth animations
- **Dynamic Navigation** - Sticky navbar with active section highlighting
- **Counter Animation** - Animated statistics on the hero section
- **Contact Forms** - Quote request and newsletter subscription forms
- **Notification System** - Toast notifications for user feedback

### Performance Optimizations
- **Lazy Loading** - Images load on demand for better performance
- **Debounced/Throttled Events** - Optimized scroll and resize handlers
- **Hardware Acceleration** - CSS transforms for smooth animations
- **Loading Screen** - Elegant loading animation

### Premium Features
- **Particle Background** - Animated particles on hero section
- **Gradient Overlays** - Modern gradient effects throughout
- **Smooth Scrolling** - Anchor links with smooth scroll behavior
- **WhatsApp Integration** - Floating WhatsApp button for easy contact
- **Back to Top** - Smooth scroll to top functionality
- **Social Media Links** - Connected social media profiles

## 🚀 Getting Started

### Option 1: Direct Opening
Simply open `index.html` in your web browser.

### Option 2: Local Server (Recommended)
For the best experience, use a local server:

**Using Python:**
```bash
python -m http.server 8000
```

**Using Node.js (http-server):**
```bash
npx http-server
```

Then navigate to `http://localhost:8000`

## 🎨 Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2d6a4f;
    --secondary-color: #d4a574;
    --accent-color: #b7791f;
    /* ... */
}
```

### Content
- Edit text content in `index.html`
- Replace product images with your own URLs
- Update contact information in the contact section

### Products
Add new products by duplicating a `.product-card` div in `index.html`
